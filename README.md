# SARTHI ELECTRICALS ⚡

**Professional Electrical Services in Gharaunda, Haryana**

A modern, responsive business website built for SARTHI ELECTRICALS — a trusted electrical engineering service provider. The site showcases services, client testimonials, and contact information with a clean, premium design.

---

## ✨ Features

- **Responsive Design** — Fully adaptive layout for desktop, tablet, and mobile
- **Smooth Animations** — Scroll-triggered reveal animations and custom smooth scrolling
- **Animated Counters** — Stats section with live number counting on scroll
- **Mobile Navigation** — Hamburger menu with smooth toggle
- **Scroll Progress Bar** — Visual scroll indicator at the top of the page
- **Back to Top** — Smooth scroll-to-top button
- **WhatsApp Float** — Quick call button for mobile users

## 🛠 Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, gradients, grid, flexbox, animations
- **JavaScript** — Intersection Observer, DOM manipulation, requestAnimationFrame

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Start the dev server
npm start

# Or open index.html directly in a browser
```
